01

Multiple Colors

• Colors - The user has access to as many as hundred of different colors on the canvas to draw.

02

Multiple In-built Shapes

• In-built Shapes - The canvas is provided with multiple shapes such as circle, rectangle etc which can be used in case of drawing.

03

Fill With Color

• Fill Colour - The shapes drawn can be filled with colours using this option. It makes those shapes look more attractive.

• Brush option - This option can be used in case of freehand drawing on the canvas. It is also available in multiple colours.

04

Eraser

• Eraser option - As the name suggest, this option is used to remove the pre-drawn works if that is not appropriate as per the user. This can erase the brush work as well as the shapes.

05

Clear

• Clear option - This option creates a new canvas by erasing the old canvas completely.

06

Save

• Save option - The user can save/download his/her drawn files using the save option. The picture is saved in jpg format.
